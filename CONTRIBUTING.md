# Contributing

This repository is a structured PCM advanced study-notes archive.

## File naming

Use:

```text
week-XX/day-YY-topic-name.md
```

Example:

```text
week-01/day-06-ode-rotation-thermodynamics.md
```

## Required daily note sections

Each completed day should include:

1. Plan row and completion status
2. Detailed subject-wise readings
3. Formula sheet
4. Important derivations
5. Solved examples and corrections
6. JEE Advanced / Olympiad practice record with difficulty ratings
7. Short ways and memory hooks
8. Common traps
9. Required output
10. Completion checklist

## Study rule

Do not mark a day complete until every topic/subtopic in that day’s plan row has been covered.
