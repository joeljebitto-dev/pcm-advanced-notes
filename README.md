# PCM Advanced Notes Repository

This repository tracks the advanced-only integrated PCM walkthrough currently being built from the uploaded 8-month Class XI-XII PCM plan.

Goal: each covered day gets a complete engineering-style study note containing:

- Summary
- Deep concepts and connected topics
- Formula sheet
- Short ways to understand the topic
- Important derivations for JEE Advanced and CBSE boards
- JEE Advanced / Olympiad-level practice record
- Completion checklist

## Operating rules

1. No day is complete until every topic in the plan row is covered.
2. Every practice problem must include a difficulty rating.
3. Practice problems should be JEE Advanced / Olympiad level unless easier warm-up is explicitly requested.
4. Each note must include CBSE-board-useful derivations where applicable.
5. New material should be appended after each covered day using the same template.
