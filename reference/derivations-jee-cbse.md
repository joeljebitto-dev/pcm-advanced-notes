# Important Derivations for JEE Advanced and CBSE Boards

## 1. Work-energy theorem

```math
F=ma
```

```math
a=v\frac{dv}{dx}
```

```math
Fdx=mvdv
```

```math
\int Fdx=\int mvdv
```

```math
W=\frac12mv_f^2-\frac12mv_i^2
```

```math
W_{net}=\Delta K
```

## 2. Spring potential energy

```math
F=-kx
```

```math
F=-\frac{dU}{dx}
```

```math
\frac{dU}{dx}=kx
```

```math
U=\frac12kx^2
```

## 3. Projectile range

```math
T=\frac{2u\sin\theta}{g}
```

```math
R=u\cos\theta \cdot T
```

```math
R=\frac{u^2\sin2\theta}{g}
```

## 4. Error propagation

For:

```math
Q=a^p b^q
```

```math
\ln Q=p\ln a+q\ln b
```

```math
\frac{\Delta Q}{Q}=|p|\frac{\Delta a}{a}+|q|\frac{\Delta b}{b}
```

## 5. Bohr transition energy

```math
E_n=-\frac{13.6}{n^2}\text{ eV}
```

```math
\Delta E=E_f-E_i
```

## 6. Bond order

```math
BO=\frac{N_b-N_a}{2}
```

Higher bond order implies shorter and stronger bond.
