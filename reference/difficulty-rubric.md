# Difficulty Rubric

| Rating | Meaning |
|---|---|
| JEE Adv L1 | Direct advanced concept application; one main idea. |
| JEE Adv L2 | Multi-step problem with two or more linked ideas. |
| JEE Adv L3 | Hard JEE Advanced style; non-obvious setup, algebra, or interpretation. |
| Olympiad L1 | Beyond standard JEE; proof-like or model-based. |
| Olympiad L2 | Deep derivation, symbolic reasoning, or nonstandard connection. |
| Olympiad L3 | Very hard; requires structural insight, barrier reasoning, higher-order tests, or abstract modelling. |

All future problems in the study notes should be tagged with one of these labels.
