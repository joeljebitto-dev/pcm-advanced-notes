

## Calculus

```math
f'(x)=\lim_{h\to0}\frac{f(x+h)-f(x)}{h}
```

```math
\frac{d}{dx}x^n=nx^{n-1}
```

```math
\frac{d}{dx}[u(x)]^n=n[u(x)]^{n-1}u'(x)
```

```math
\int x^n dx=\frac{x^{n+1}}{n+1}+C,\quad n\ne -1
```

```math
\int \frac1x dx=\ln|x|+C
```

```math
\frac{d}{dx}\int_a^{g(x)}f(t)dt=f(g(x))g'(x)
```

```math
\frac{d}{dx}\int_{a(x)}^{b(x)}f(t)dt=f(b)b'-f(a)a'
```

```math

```

```math

```


```math

```

## Periodicity

```math
Z_{eff}=Z-S
```

Across period: radius decreases, ionization energy increases, electronegativity increases.

Down group: radius increases, ionization energy decreases, electronegativity decreases.
