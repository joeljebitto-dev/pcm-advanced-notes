## Key formulas

### Exponentials and logs

```math
a^x=y \iff \log_a y=x
```

```math
\ln(e^x)=x
```

```math
e^{\ln x}=x
```

Scaling:

```math
y\propto x^n,\quad x\to kx \Rightarrow y\to k^n y
```

### Kinematics

```math
v(t)=\frac{dx}{dt}
```

```math
a(t)=\frac{dv}{dt}=\frac{d^2x}{dt^2}
```

If `a(t)=6` and `v(0)=2`:

```math
v(t)=6t+2
```

If `x(0)=1`:

```math
x(t)=3t^2+2t+1
```

### Bohr energy levels

```math
E_n=-\frac{13.6}{n^2}\text{ eV}
```

```math
\Delta E=E_f-E_i
```

Examples:

```math
E_1=-13.6\text{ eV}
```

```math
E_2=-3.4\text{ eV}
```

```math
E_3\approx -1.51\text{ eV}
```

## Short ways to understand

- A log asks: “what exponent produced this?”
- Exponential and logarithmic functions are inverse maps.
- Velocity is the derivative of position; acceleration is the derivative of velocity.
- Bohr levels become less negative as `n` increases; higher `n` means higher energy.
