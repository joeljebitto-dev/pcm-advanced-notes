## Summary

Day 3 connected trigonometry, vectors, complex numbers, projectile motion, and orbitals. The common structure is orientation: angles in math, direction in physics, and quantum orientation in chemistry.

## Key formulas

### Radians and Euler identity

```math
180^\circ=\pi\text{ rad}
```

```math
e^{i\theta}=\cos\theta+i\sin\theta
```

```math
z=a+bi=re^{i\theta}
```

```math
|z|=\sqrt{a^2+b^2}
```

### Vectors

```math
|\vec A|=\sqrt{A_x^2+A_y^2}
```

```math
\vec A\cdot\vec B=A_xB_x+A_yB_y=AB\cos\theta
```

```math
|\vec A\times\vec B|=AB\sin\theta
```

Unit cross products:

```math
\hat i\times\hat j=\hat k
```

```math
\hat j\times\hat i=-\hat k
```

### Projectile motion

```math
u_x=u\cos\theta
```

```math
u_y=u\sin\theta
```

```math
x(t)=u_x t
```

```math
y(t)=u_y t-\frac12gt^2
```

```math
T=\frac{2u\sin\theta}{g}
```

```math
R=\frac{u^2\sin2\theta}{g}
```

```math
H=\frac{u^2\sin^2\theta}{2g}
```

### Quantum numbers

```math
n=1,2,3,\dots
```

```math
l=0,1,\dots,n-1
```

```math
m_l=-l,\dots,+l
```

```math
m_s=\pm\frac12
```

```math
\text{orbitals}=2l+1
```

```math
\text{max electrons}=2(2l+1)
```

## Electronic configurations covered

```text
C  = 1s^2 2s^2 2p^2
O  = 1s^2 2s^2 2p^4
Na = 1s^2 2s^2 2p^6 3s^1
Cl = 1s^2 2s^2 2p^6 3s^2 3p^5
```

## Short ways to understand

- Complex numbers behave like 2D vectors with multiplication as rotation-scaling.
- Dot product measures alignment.
- Cross product measures perpendicular area and orientation.
- Projectile motion is two independent motions: uniform horizontal and accelerated vertical.
- Quantum numbers label shell, shape, orientation, and spin.

## Important derivations

### Projectile range

For launch and landing at same height:

```math
T=\frac{2u\sin\theta}{g}
```

Horizontal distance:

```math
R=u\cos\theta\cdot T
```

```math
R=u\cos\theta\frac{2u\sin\theta}{g}
```

```math
R=\frac{u^2\sin2\theta}{g}
```
